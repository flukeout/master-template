Intermediate Web Literacy: CSS I - CSS Pyramid

Made by Mozilla Leadership Network

Learners will use CSS to change the size of text on an HTML webpage, learning about coding, composing, and design.

60 minutes

For beginners

Learning Objectives

  • Understand and be able to explain the connection between an HTML webpage and a CSS stylesheet.
  • Change the size of text on an HTML webpage using selectors and values on a CSS stylesheet.

Audience

  • 13+
  • Intermediate web users

Materials

  • Internet-connected computers
  • Markers
  • Dino drawing styling instructions template
